# Kuria

A personal finance app for multi-currency tracking, budgeting, and investment management. Built on Rails 8.

## Features

- **Account tracking** — bank accounts, credit cards, investments, loans, properties, vehicles, and crypto
- **Multi-currency support** — track balances and transactions across currencies with exchange rate data
- **Transaction management** — categorization, tagging, bulk editing, and auto-classification rules
- **Budgets** — monthly budget planning and tracking
- **Investment portfolio** — holdings, valuations, and performance tracking
- **AI financial assistant** — conversational assistant with access to your financial data
- **Imports** — CSV import, Plaid bank sync, and Mint data migration
- **Data exports** — export your data at any time
- **Self-hostable** — run it yourself with Docker

## Local Development Setup

### Requirements

- Ruby 4.0.2 (see `.ruby-version`)
- Docker (for running dependencies)

### Setup

Start the infrastructure dependencies (Postgres + Redis) with Docker:

```sh
docker compose -f compose.dev.yml up -d
```

Then set up and run the Rails app locally:

```sh
git clone https://github.com/petebarbosa/maybe.git
cd maybe
cp .env.dev .env.local
bin/setup
bin/dev
```

To stop the dependencies:

```sh
docker compose -f compose.dev.yml down
```

Visit [http://localhost:3000](http://localhost:3000). Default credentials generated by the seed:

- Email: `user@maybe.local`
- Password: `password`

### Demo data

```sh
rake demo_data:default
```

### Tests

```sh
bin/rails test          # unit and integration tests
bin/rails test:system   # browser-based system tests
```

## Self-Hosting with Docker

See [`docs/hosting/docker.md`](docs/hosting/docker.md) for the full guide.

At minimum you will need to set the following environment variables:

| Variable | Description |
|---|---|
| `SECRET_KEY_BASE` | Required. Generate with `openssl rand -hex 64`. |
| `SELF_HOSTED` | Set to `true` to enable self-hosting mode. |
| `SYNTH_API_KEY` | Exchange rate and market data (optional). |
| `OPENAI_ACCESS_TOKEN` | Required for the AI assistant feature (optional). |
| `SMTP_*` | Mail delivery configuration (optional). |
